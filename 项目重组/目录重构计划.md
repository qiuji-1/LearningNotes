# 目录重构计划

## 目标
将项目重新组织为更清晰的结构，同时清理Git历史中不应该追踪的文件（如node_modules）。

## 新的目录结构

```
source code/
├── .git/                 # Git 版本控制
├── .gitignore           # Git 忽略规则
├── .vscode/             # VS Code 项目配置（根目录）
│   ├── settings.json    # 编辑器设置
│   └── launch.json      # 调试配置
├── src/                 # 源代码目录（新建）
│   ├── api/             # SpringBoot 后端
│   ├── view/            # Vue 前端
│   └── sql/             # 数据库脚本
├── config/              # 配置文件目录（新建）
│   ├── .vscode/         # 备份配置
│   └── god key/         # 密钥配置
├── docs/                # 文档目录
│   ├── 对话记录/        # 工作日志
│   ├── 论文/            # 毕业论文
│   ├── Learning/        # 学习资料
│   └── 个人计划表/      # 个人计划
├── temp/                # 临时文件目录
└── README.md            # 项目说明文档
```

## 配置文件更新

### 1. 新建 `.vscode/settings.json`（根目录）
```json
{
    "java.compile.nullAnalysis.mode": "automatic",
    "java.project.sourcePaths": [
        "src/api/src/main/java"
    ],
    "java.project.outputPath": "src/api/target/classes",
    "java.configuration.runtimes": []
}
```

### 2. 新建 `.vscode/launch.json`（根目录）
```json
{
    "version": "0.2.0",
    "configurations": [
        {
            "type": "java",
            "name": "Launch SpringBoot App",
            "request": "launch",
            "mainClass": "com.kmbeast.SelfHealthApiApplication",
            "projectName": "self-health-api",
            "cwd": "${workspaceFolder}/src/api",
            "args": "",
            "vmArgs": ""
        }
    ]
}
```

### 3. 更新 `README.md`
- 更新项目结构说明
- 更新启动命令（cd src/api, cd src/view）

## 启动命令调整

### 重构前
```bash
# 后端
cd api
mvn spring-boot:run

# 前端
cd view
npm run serve

# 数据库
mysql -u root -p < sql/health.sql
```

### 重构后
```bash
# 后端
cd src/api
mvn spring-boot:run

# 前端
cd src/view
npm run serve

# 数据库
mysql -u root -p < src/sql/health.sql
```
