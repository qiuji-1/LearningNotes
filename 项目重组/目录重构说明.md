## 对项目运行的影响

### 不影响运行的原因
- SpringBoot 配置文件使用相对路径，不受目录移动影响
- Vue 配置使用相对路径和端口号，不受影响
- 数据库脚本是独立文件，执行时指定路径即可

### 可能需要调整的地方
1. **VS Code 工作区**：需要重新加载 VS Code 窗口（Ctrl+Shift+P -> Reload Window）
2. **启动命令**：
   - 后端：`cd src/api` 然后 `mvn spring-boot:run`
   - 前端：`cd src/view` 然后 `npm run serve`
3. **数据库脚本**：`mysql -u root -p < src/sql/health.sql`
