## 🚀 如何使用

### 编译运行单个文件

**Windows 系统（需指定 UTF-8 编码）：**

```bash
# 编译
javac -encoding UTF-8 Question01_ForForeach.java

# 运行
java Question01_ForForeach
```

**Linux/Mac 系统：**

```bash
# 编译
javac Question01_ForForeach.java

# 运行
java Question01_ForForeach
```

### 编译运行所有文件

**Windows 系统：**

```bash
# 编译所有文件
javac -encoding UTF-8 *.java

# 运行指定文件
java Question01_ForForeach
```

**Linux/Mac 系统：**

```bash
# 编译所有文件
javac *.java

# 运行指定文件
java Question01_ForForeach
```

## 📝 注意事项

1. 所有测试代码均可独立运行
2. 运行前请确保已安装 JDK 8 或更高版本
3. 部分代码可能需要 Java 8+ 特性（如 Lambda 表达式、Stream API）
4. **Windows 系统编译时需要指定 UTF-8 编码**：`javac -encoding UTF-8`
5. 如果代码包含中文注释，推荐使用 UTF-8 编码保存
