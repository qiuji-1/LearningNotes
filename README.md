# 学习资料仓库

这个仓库用于存放学习和个人管理相关的资料，与项目代码分开管理。

## 目录结构

```
learning-notes/
├── docs/                   # 文档目录
│   ├── Learning/          # 学习资料
│   │   ├── LearnGit/   # Git 学习
│   │   ├── LearnJava/  # Java 学习
│   │   └── LearnSSH/   # SSH 学习
│   └── 对话记录/         # 工作日志和对话记录
├── 个人计划表/            # 个人计划
└── README.md             # 说明文档
```

## 内容说明

### docs/Learning/
学习笔记和代码示例，包括：
- Git 使用技巧和教程
- Java 核心特性学习（反射、序列化等）
- SSH 配置和使用

### docs/对话记录/
工作日志和对话记录，记录：
- 每天的工作进展
- 学习心得和问题解决
- CodeBuddy 对话记录

### 个人计划表/
个人时间管理文件：
- 月度计划
- 完成情况记录

## 与主项目的关系

这个仓库是 `source code` 仓库的补充：

- **source code**：只存放毕业项目代码（SpringBoot + Vue）
- **learning-notes**：存放学习资料和个人管理文件

这样分离可以：
- ✅ 保持项目仓库简洁
- ✅ 避免追踪不必要的文件（如 Java 学习的编译输出）
- ✅ 更好地管理个人学习和项目开发
